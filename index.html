Oh hi!

I was born and raised in Vancouver. I consider myself a designer and front-end developer but I definitely lean more towards designer.

I started off my career as a web designer and I'm very comfortable with html/css. A few years ago I decided I wanted to be able to do more than simply modify existing jquery/javascript plugins. I was frustrated because I'd find a plugin that would do 90% of the things I wanted but that last 10% "should" be an easy addition but not for me. So I went back to BCIT and took a few programming courses (intro to web dev with java 1 & 2 and c++ 1). This helped a lot but I know that I still have a ways to go.

I wouldn't consider myself a javascript developer though, I'd place myself somewhere between novice and intermediate. I love being able to switch between design and development to keep my days intersting and I find it really helps with my ability to communicate with developers.
